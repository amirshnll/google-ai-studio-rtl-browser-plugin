# Google AI Studio RTL

This extension has a single purpose: to improve the readability and usability of Google AI Studio for users who need Right-to-Left (RTL) text support and font customization. It applies local interface adjustments such as RTL layout, text direction, and selected fonts only on aistudio.google.com, without affecting other websites or collecting user data.

**Install:** https://chromewebstore.google.com/detail/google-ai-studio-rtl/fclfjblilgjmanaeidbgahelpppgghpl
